# 投资博客 - 操作指南

> 原文链接: https://github.com/xiwenkaka/invest-blog

## 第一步：配置 GitHub（只需做一次）

### 1.1 创建博客仓库
1. 打开 https://github.com/new
2. Repository name 填：`invest-blog`
3. 选择 **Public**
4. 点 **Create repository**

### 1.2 配置 GitHub Pages
1. 进入你刚创建的仓库
2. 点 **Settings** → 左侧 **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选择 **gh-pages** → **/ (root)** → 点 **Save**
5. 等2分钟，博客地址就变成：`https://xiwenkaka.github.io`

### 1.3 生成 GitHub Token（自动化部署用）
1. 打开 https://github.com/settings/tokens
2. 点 **Generate new token (classic)**
3. Note 填：`blog-deploy`
4. 勾选 `repo` 和 `workflow`
5. 点 **Generate token**
6. **把生成的 Token 复制保存好**（只显示一次！）

### 1.4 添加 Token 到仓库
1. 回到你的 `invest-blog` 仓库
2. 点 **Settings** → 左侧 **Secrets** → **Actions**
3. 点 **New repository secret**
4. Name 填：`GITHUB_TOKEN`
5. Secret 填：你刚复制的 Token
6. 点 **Add secret**

### 1.5 添加 Tushare Token（如果你要用股票数据）
1. 同样在 Secrets 页面
2. 点 **New repository secret**
3. Name 填：`TUSHARE_TOKEN`
4. Secret 填：你的 Tushare Pro Token（之前配置过）
5. 点 **Add secret**

## 第二步：上传博客代码

### 2.1 下载代码包
我会把所有文件打包，你下载后会上传到你刚创建的 GitHub 仓库。

### 2.2 上传到 GitHub
**方法A：网页上传（最简单）**
1. 打开你的 GitHub 仓库
2. 点 **Add file** → **Upload files**
3. 把所有文件拖进去
4. 点 **Commit changes**

**方法B：用 Git 命令行**
```bash
git clone https://github.com/xiwenkaka/invest-blog.git
cd invest-blog
# 把下载的文件复制进来
git add .
git commit -m "init blog"
git push
```

## 日常使用

### 写文章
1. 在 `source/_posts/` 目录新建 `.md` 文件
2. 格式参考 `sample-post.md`
3. 写完 push 到 GitHub，自动发布

### 更新数据页面
1. 在 GitHub 仓库点 **Actions**
2. 点 **Auto Update Blog**
3. 点 **Run workflow** → **Run workflow**
4. 等1分钟，数据页面就自动更新了

### 手动更新数据（本地）
```bash
# 安装依赖
npm install
pip install tushare pandas requests

# 设置 Token
export TUSHARE_TOKEN="你的Token"

# 运行更新
npm run update-all
```

## 常见问题

**Q: 博客打不开？**
A: 等2分钟，GitHub Pages 需要时间生效。确认 Settings → Pages → Source 是 gh-pages。

**Q: 数据没更新？**
A: 去 Actions 页面看运行日志，红色是失败了。

**Q: 想改博客名字？**
A: 编辑 `_config.yml` 文件，push 后自动生效。
